# Taking input
## Easy 
<div class="problem-statement">
                <p></p><p><span style="font-size:18px">Before implementing any algorithm, we should be thorough with taking inputs. Here, we will learn how to read inputs.<br>
You are given two inputs: <strong>a(integer), and b(string)</strong>. You need to take the input and print a and b separated by a space.</span></p>

<p><span style="font-size:18px"><strong>Example 1:</strong></span></p>

<pre><span style="font-size:18px"><strong>Input:</strong>
a = 5
b = Hello
<strong>Output:
</strong>5 Hello</span></pre>

<p><span style="font-size:18px"><strong>Example 2:</strong></span></p>

<pre><span style="font-size:18px"><strong>Input:</strong>
a = 7
b = Geeks
<strong>Output:
</strong>7 Geeks</span></pre>

<p>&nbsp;</p>

<p><span style="font-size:18px"><strong>Constraints:</strong><br>
1 &lt;= a &lt;= 10<sup>6</sup></span></p>
